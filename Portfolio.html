About Me

I am Hemanth Dorepalli, a passionate software developer with a Bachelor of Technology, eager to pursue a full-stack web developer role where I can bring my skills and creativity to the forefront. I have experience with both front-end and back-end technologies, including Java, JavaScript, ReactJS, and Spring Boot. My notable projects include a responsive Chatting Web Application developed using React Js, HTML, and CSS, with Firebase integration for real-time messaging, user authentication, and profile management. I also created a Weather Dashboard with React Js, HTML, Tailwind CSS, and JavaScript, integrating the OpenWeatherMap API for real-time weather data. Additionally, I developed the My Diary Web Application using Spring Boot and Thymeleaf, which features secure user authentication and diary management, and is deployed on AWS for reliable access. These projects reflect my ability to build effective, user-centric solutions and my enthusiasm for contributing to innovative and collaborative environments.
